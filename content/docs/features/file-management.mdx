The File Browser lets you manage your game server's files directly from the COSY web interface — no SSH or FTP access required.

## Features

- **Directory navigation** — Browse the full file tree of your server's volumes
- **File preview** — View text files, images, and videos inline
- **Upload** — Upload files and folders to your server
- **Download** — Download individual files or entire directories as ZIP archives
- **File operations** — Create, rename, move, and delete files and directories
- **Permission display** — View Unix file permissions

## Common Use Cases

### Installing Mods and Plugins

Upload mod or plugin files to the appropriate directory:

1. Navigate to the mods/plugins directory (e.g., `/data/mods` for Minecraft Forge)
2. Upload your `.jar` files
3. Restart the server for the changes to take effect

### Downloading Backups

Download your world save or entire server directory as a ZIP:

1. Navigate to the directory you want to download
2. Right-click and select download, or use the download button
3. The directory is compressed and downloaded to your browser

## Required Permissions

To access the File Browser through [Access Groups](/docs/features/access-management):

| Permission                 | Access                       |
| -------------------------- | ---------------------------- |
| `READ_SERVER_SERVER_FILES` | Browse and download files    |
| `CHANGE_SERVER_FILES`      | Upload, rename, delete files |
